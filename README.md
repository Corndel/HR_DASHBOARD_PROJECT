# HR Analytics Dashboard

This project is an SAP-style HR analytics dashboard developed for the DT402 module.

## Overview
The dashboard provides insights into workforce composition, performance, training compliance, and absence trends using fictional HR data.

## Technologies Used
- Python (pandas) for data cleaning and KPI calculation
- HTML and CSS for structure and layout
- JavaScript (Chart.js) for interactive visualisations
- Git and GitHub for version control

## How to Run
1. Run the Python scripts in the scripts folder to generate cleaned data.
2. Open frontend/index.html in a browser to view the dashboard.

## Notes
All data used is fictional and for academic purposes only.
